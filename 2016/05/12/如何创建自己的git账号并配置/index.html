如何创建自己的git账号并配置

创建GitHub账号

注册GitHub账号，只要点击导航条中的“Signup and Pricing”，或者点击首页中那个大大的“Plans, Pricing and Signup”按钮，即进入收费方案介绍及注册页面。点击免费方案右侧的“Create a free account”按钮，就进入到注册页面，如图所示 请注意是以您的用户名开头的,例如我的为： “ CracKerMe.github.io ” 用户名后面的“.github.io”不得修改，此为您的二级域名。




配置github


1.首先建立一个仓库，仓库名必须与你的用户的名对应，就是仓库名必须是“your-name.github.io”


2）设置用户名和email。

git config –global user.name “Your Name”

git config –global user.email “youremail@domain.com”

3.为github账号添加ssh keys

3.1 ）创建SSH key 在用户界面上选择 Settings ，选择 SSH and GPG keys



在终端中输入 ssh-keygen 3个回车，跳出一长串代码
这个时候输入“vim ~/.ssh/id_rsa.pub ”

此时复制 ssh开头到local 的所有字符串，复制好了之后，填入 上面的 key 里面。 Add SSH key!

不过，小伙伴们是不是 卡在刚才的界面，不得动弹？按下 Shift + : + q

最后就剩下上传了
